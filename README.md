# 명철호 포트폴리오

DirectX11 기반 자체 게임 엔진, Unreal 엔진을 활용한 3D 액션 RPG,  
Unity 엔진을 활용한 2D 퍼즐 게임 & 3D 액션 RPG, Socket API 기반 서버 개발 경험이 있습니다.

기능이 되는 것에서 멈추지 않고,  
수치로 검증하고 결과를 확인하는 과정을 중요하게 생각합니다.  
항상 더 나은 구조를 고민하고 학습하며, 리팩터링을 통해 반복 작업을 줄이고  
유지보수성과 확장성을 높이는 것을 지향합니다.

어려운 문제를 마주하면 여러 가설을 세우고 끝까지 추적해 해결하는 편입니다.  
이 과정에서 얻어지는 이해와 경험이 비로소 제 것이 된다고 믿기 때문에,  
문제를 피하기보다 원인 단위로 파고드는 태도를 중요하게 생각합니다.

팀 프로젝트에서는 팀장 역할을 맡아  
팀원 간 소통을 돕고, 역할 분담 & 일정 조율을 담당한 경험이 있습니다.  
이를 통해 의견을 조율하고, 팀이 같은 방향을 보도록 만드는 데 강점을 갖게 되었습니다.
